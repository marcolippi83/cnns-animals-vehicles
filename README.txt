Download models from the following link:
https://drive.google.com/drive/folders/1UUxSXNmgacavnsxessuaD8unBIQYBU2-
and place them into the models folder.

In the scripts folder, use the bash scripts predict.sh and predict_original.sh to classify test images with fine-tuned and original models, respectively. It might be necessary to edit the bash script to update the value of some variables (for example, python version).

Prerequisites:
- Keras (version 2.1.4)
- Tensorflow (version 1.4.1)
